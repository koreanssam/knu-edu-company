## 4. Lesson Improvement Proposal Generation
1. Identify key issues from analysis results:
   - Learning objectives with low accuracy rates.
   - Areas where misconception clusters were found.
   - Items with low discrimination (suggesting insufficient teaching effectiveness).
2. Propose supplemental learning materials:
   - Explanation strategies for correcting misconceptions.
   - Additional practice problem types.
   - Visual/experiential learning activities.
3. Next lesson activity design:
   - Review activities (for weak areas).
   - Enrichment activities (for advanced student groups).
   - Collaborative learning opportunities.
4. Personalized learning paths by student group:
   - High: Enrichment tasks, challenge problems.
   - Mid: Reinforcement of core concepts.
   - Low: Basic supplementation, individual guidance points.

## 5. Deliverables
- `improvement-proposal.md`:
  - Summary (Key findings + recommendations).
  - Supplemental materials list.
  - Draft of next lesson plan.
  - Learning paths by student group.
  - Impact/Execution difficulty indicators for each recommendation.

## 6. Approval Request
- `@edu-ops-supervisor`: "Lesson improvement proposal draft complete. Requesting teacher review."
- Specify approval gates for suggestions including curriculum changes.
