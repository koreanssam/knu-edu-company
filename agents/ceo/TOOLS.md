# CEO Tools

**Available Skills:**

* `paperclipai/paperclip/paperclip` — Paperclip core. Agent coordination, issue management, approval workflow.
* `paperclipai/paperclip/para-memory-files` — Persistent memory. Record operational decisions, loop progress, and teacher feedback.
* `paperclipai/paperclip/paperclip-create-agent` — Create new agents. Use when manpower is insufficient.
* `knu/edu-shared` — Learning Operations Center common rules. Data schema, security rules, deliverable standards.

**Core Reference Files:**

* `/AGENTS.md` — Learning Operations Center Operating Guidelines (Common to all agents)
* `TEACHING_BRIEF.md` — Learning goals and teaching materials set by the teacher (Project root)
* `$AGENT_HOME/memory/` — Daily plans, progress records, decision logs

**Execution Loop:**
Follow the checklist in HEARTBEAT.md. Track progress between sessions using para-memory-files.
