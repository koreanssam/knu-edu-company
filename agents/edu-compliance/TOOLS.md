# Edu Compliance Officer Tools

## Skills
- **paperclip** — Issue management, escalation.
- **para-memory-files** — Record audit logs, security patterns.
- **edu-shared** — Personal information rules, security policies.

## Key Reference Files
- `/AGENTS.md` — Operating Guidelines (Prohibitions, notification steps, Data Rules).
- `$AGENT_HOME/audit-log/` — Audit log directory.

## Inspection Targets
- Email body and recipient list.
- Form/Sheet sharing settings.
- Personal information within attachments.
- Anonymization status of analysis reports.
- External sharing link scope.

## Deliverables
- Security check result report (Issue comment).
- Audit log entry.
- Block/Warning notifications.
