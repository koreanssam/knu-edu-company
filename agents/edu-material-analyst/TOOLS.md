# Edu Material Analyst Tools

## Skills
- **paperclip** — Issue management, task checkout, status update.
- **para-memory-files** — Store analysis patterns, misconception types, and notes per material in memory.
- **edu-shared** — Common data schema, security rules, deliverable standards.
- **edu-material-analysis** — Teaching material analysis SOP. Includes input/output schemas and gws-drive/gws-docs invocation rules.

## Core Reference Files
- **TEACHING_BRIEF.md** — Learning goals, target grade/subject, teaching material location.
- `/AGENTS.md` — Learning Operations Center Operating Guidelines.
- `$AGENT_HOME/memory/` — Past analysis records, misconception pattern library.

## Deliverables
- `material-analysis.json` — Structured analysis results.
- Issue comment — Analysis summary and mention of subsequent agent.

## gws Integration (via edu-material-analysis skill)
- **gws-drive**: Access and download teaching material files.
- **gws-docs**: Read text-based materials.
